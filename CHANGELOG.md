# 更新日志（Changelog）

本项目的版本变更记录。版本号遵循 `主.次.修订`。

## v1.0.5

- **新增：ContinuousUpdates（连续更新）伪编码**。客户端协商 `-313`；服务器支持时回发
  `EndOfContinuousUpdates`，客户端开启后由**服务器主动推送增量帧**，不再逐帧发送
  `FramebufferUpdateRequest`，**省掉每帧一次往返延迟、更跟手**。服务器不支持时自动回退到
  原"请求-应答"循环，零副作用。

## v1.0.4

- **修复：工作一段时间后顶部工具栏（菜单）消失、点击无反馈**。
  - 移除视口的鼠标捕获（`CaptureMouse`）——捕获一旦卡住会让工具栏触发区收不到鼠标、
    还会拦截工具栏按钮点击。改为每次移动从事件参数同步真实按钮状态（自动纠正漏掉的
    按下/松开）、离开视口即释放远端按钮。
  - 消除工具栏 XAML 触发器与代码后台的双重动画打架（会卡在 `Opacity=0` 看着像消失），
    改为确定性显示/隐藏；鼠标停在工具栏上时不自动隐藏，方便点按钮。
- **修复：本地复制内容无法在 Mac 粘贴的一处隐患**。收发两方向原共用 `_isSyncingClipboard`
  标志，macOS 频繁回发剪贴板会把发送方向卡死；改为单一 `_lastClipboard` 值判重，天然
  防回声、无竞态。

## v1.0.3

- **修复：鼠标操作不丝滑、工作几分钟后所有点击无反馈（卡死）**。
  - 输入写入改为**后台写线程 + 写队列**：UI 线程只入队不阻塞，避免发送缓冲填满时
    同步写阻塞 UI 线程导致界面冻结。
  - socket 增加 **SendTimeout(5s) + TCP keepalive(15/5/3s)**：半死连接约 30 秒内被发现 →
    触发自动重连恢复。
  - 鼠标移动**按远程像素+按钮状态去重**，减少洪泛、更跟手。

## v1.0.2

- **修复：退出后立即重连被 macOS 重置**（`远程主机强迫关闭了一个现有的连接`）。
  退出后过快重连时 macOS 上一会话（screensharingd）尚未释放会 RST。连接序列改为
  最多 5 次、递增退避（约 3/5/8/11 秒，共 ~27 秒）的重试，不猛连，覆盖释放窗口；
  认证失败/超时不重试。

## v1.0.1

- **新增：Apple/ARD 认证（安全类型 30）**——macOS 屏幕共享默认认证，支持用户名+密码
  （DH + MD5 + AES-128-ECB）。
- **新增：远端光标本地渲染**（Cursor 伪编码 -239），鼠标更跟手、无重影。
- **新增：剪贴板 UTF-8 自适应编码**，改善 Mac↔Windows 中文/Emoji 剪贴板。
- **增量刷新**：只拷贝/重绘变化矩形，大幅提升流畅度；合并式 `BeginInvoke` 渲染。
- **韧性**：断线自动重连（含快速掉线保护）、仅查看模式、画质（16/32 位色）选项。
- **深度多智能体审计修复**一批输入/编码/安全问题：Alt/Option 键、Shift 先松导致远端卡键、
  Hextile 彩色子矩形颜色泄漏、DES/DPAPI 密码加解密损坏、连接失败 NRE、整数溢出防护、
  配置原子写等。
- 主窗口标题栏显示版本号。

## v1.0.0

- 首个可用版本：从无法编译的初始代码完成整体修复与重写，可在真实 Windows 上构建运行。
- VNC 客户端核心：RFB 3.8 握手、VNC 密码(DES)/无认证、初始化协商。
- 编码：Raw、Hextile、CopyRect、ZRLE（帧缓冲统一 BGRA32）。
- 连接管理器（连接簿、快速连接、DPAPI 加密保存密码）、远程会话窗口（全屏/缩放、
  Mac 快捷键菜单、文本剪贴板同步）。
- 发布：win-x64 单文件自包含 `MiniVNC.exe`，目标机无需预装 .NET 运行时。
